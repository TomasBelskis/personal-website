Final Year Project


  • JavaScript
  • NodeJS
  • Bootstrap
  • WebRTC API
  • Kurento MS API

My FYP consisted of research based on current day conference technologies. It focused on delivering an environment that could be used to carry out lessons online. It started with focus on educational content and providing an environment that allows to stream classes to multiple clients and additional features such as displaying pdf slides, group chat, and file uploading. During the development of the project it evolved into a web application that would allow individual users to host their own educational channel that would support webcam streaming and screen share stream without any additional software or plugins. A wide variety of technologies were covered throughout this project such as WebRTC browser API that allows to capture media through a browser also multiple NodeJS libraries were covered. The project was build using cloud technologies, Kurento media server and NodeJS server.

Auction Client/Server Java


  • Java
  • Client
  • Server

A simple java application that supports multiple client connections, as soon as a single client connects an auction starts, the client is able to bid for an item. There is a timer that is reset until there are bids being made, if the timer runs out that particular items is sold to the highest bidder and the auction moves on to next item in the list (also if item is not sold it moves on to next item) once all items are sold it is considered that the auctions is a success, the server shuts downs and disconnects all the clients.
